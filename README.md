# LangGraph 101
Hands-on exploration of [LangGraph](https://www.langchain.com/langgraph) for multi-step AI agent workflows. This repo tracks my progress from core concepts to a functional end-to-end project.

### Fundamentals
- LangGraph's graph-based execution model (nodes, edges, state)
- Conditional branching and cyclic workflows
- Integrating tools and external APIs into agent loops
- Managing agent memory and state across steps
